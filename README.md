# ☁ Cloud Notes Application

A responsive cloud-style notes management web application built using HTML, CSS, and JavaScript. Users can create, edit, delete, and search notes with data stored locally in the browser using Local Storage.

---

# 🚀 Features

- Create Notes
- Edit Existing Notes
- Delete Notes
- Search Notes
- Responsive Design
- Local Storage Support
- Modern UI Design
- Mobile Friendly

---

# 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Browser Local Storage
